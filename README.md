# 陈景腾的博客

### 项目描述

技术分享、经验总结、心得体会

### 主要技术

* webpack（开发环境、生产环境调试打包）

* react（构建ui界面）

* redux（管理数据）

* redux-thunk（redux的异步处理）

* redux-persist（redux数据持久化）

* react-redux（连接react和redux）

* react-router-dom（管理路由）

* react-loadable(组件异步加载)

* pm2（自动化部署到线上服务器）

### 项目运行

#### 安装依赖

推荐使用yarn，不推荐npm

    cd blog

    yarn

#### 开发环境

本地运行

    yarn start

打包，生成dist目录
	
    yarn run dev

#### 生产环境

打包，生成build目录

    yarn run build
